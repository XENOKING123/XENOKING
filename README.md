## 🎯 Cheats & Trainers — the main event

<img src="client/public/goku.jpg" align="right" width="190" alt="XENO"/>

- 📦 **All ~9,200 trainers are bundled** — no downloads, it's ready on first launch.
- 🗂️ One clean card per game, **all versions merged** onto a side rail.
- 🔌 Hit **Attach** while the game is running → the dot turns 🟢.
- 🔴➡️🟢 Toggle a cheat to apply it **live**; 🟢➡️🔴 reverts it. Sounds confirm every write.
- 🧹 **Detach** turns everything back off and disconnects cleanly.
- 🛡️ **Safe:** writes are bounded to the game's own code, size-capped, and the
  game is re-checked right before every write — it won't smear memory or crash your console.
- 🔄 Auto-syncs new cheats from the repos every **24 hours**.

## 🚀 Quick start

```text
1.  ⬇️   Download  XENO_TOOL_x.x.x_x64-setup.exe  from Releases
2.  ▶️   Run it  (SmartScreen → More info → Run anyway)
3.  🧩   Jailbreak / load your payload on the PS5 as usual
4.  🔗   Connection tab → enter your PS5's IP → Connect
5.  🎯   Trainers / My Games → Attach → flip cheats LIVE
```

> 💡 First launch unpacks the full trainer library (a few seconds) and opens on
> the **Home** showcase. 🏠

## 🛠️ Build from source

<details>
<summary><b>Click to expand build steps</b> 🧰</summary>

<br/>

```bash
# 1) frontend deps
cd client && npm install

# 2) build the engine sidecar first (it gets embedded)
cd ../engine && cargo build --release

# 3) build the desktop app + installer
cd ../client && npm run tauri build
```

The installer lands in `client/src-tauri/target/release/bundle/nsis/`.

**Requirements:** Node 20+, Rust (stable, MSVC on Windows). The bundled trainer +
title seeds live in `client/src-tauri/resources/*.zip`.

</details>

## ❓ FAQ

<details>
<summary><b>Do I need to download the trainers separately?</b></summary>
<br/>
Nope. All ~9,200 ship inside the app and unpack on first launch. It also
auto-updates from the repos every 24h.
</details>

<details>
<summary><b>SmartScreen is warning me — is it safe?</b></summary>
<br/>
The installer isn't code-signed (signing costs money), so Windows shows the
generic warning. Click <b>More info → Run anyway</b>.
</details>

<details>
<summary><b>Cheats won't apply?</b></summary>
<br/>
The game must be <b>running</b> on the PS5, and you must hit <b>Attach</b> first
(the dot goes green). If a write is refused, the toggle reverts itself.
</details>
